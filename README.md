# Temettü Takip

Tek dosyalık (`index.html`) temettü ve fon takip ekranı.

## Canlı Kullanım

Bu proje GitHub Pages üzerinden statik olarak yayınlanır.

## Özellikler

- Hisse alım kayıt yönetimi
- Temettü kayıtları ve özet görünümü
- Fon ekleme ve NAV takibi
- Modal yönetimi (`İptal` / `Kaydet` / `Esc` / dış alan tıklama)
- İşlem sonrası otomatik ekran yenileme

## GitHub Pages Yayınlama

1. Bu repoyu GitHub'a yükleyin.
2. GitHub'da repo ayarlarından `Settings > Pages` bölümüne gidin.
3. `Deploy from a branch` seçin.
4. Branch olarak `main`, klasör olarak `/ (root)` seçin.
5. Kaydedin ve yayın URL'inin oluşmasını bekleyin.

## Notlar

- Bu proje statik çalışır; ek bir backend zorunlu değildir.
- Fon bilgileri için otomatik veri çekme tarayıcı tarafında dış proxy servislerine bağlıdır.
- Bu nedenle bazı zamanlarda yavaşlık veya geçici erişim hataları görülebilir.
